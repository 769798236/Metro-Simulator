# 地铁站牌生成模拟器

## 项目说明
一个提供地铁站牌生成的模拟器

首发于地铁族,因此大小与地铁族论坛最大支持签名档大小同步(700 × 166)

## 目前状况

- 使用Material UI重构
- ~~使用TypeScript重构(天坑×1)~~
- 使人犯困的夜间模式(天坑×2)
- 图片使用SVG重绘
- 多种样式选择

### 优先处理

- 右边药丸边框缺失问题

- 组件位置重构下

- 数据使用Vuex存储

## 离线测试步骤

请确定您的计算机已安装Node.js

打开命令行模式,键入命令

> git clone https://github.com/iMisty/Metro-Simulator.git

> cd Metro-Simulator

> npm install

Enjoy
